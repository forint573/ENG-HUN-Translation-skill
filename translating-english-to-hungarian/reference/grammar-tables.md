# Hungarian grammar tables (reference)

Exhaustive paradigms and inventories backing `../SKILL.md`. Consult a section only when you need a full table; the rules and the high-frequency forms are already in SKILL.md. All forms here are spelled with their correct accents and obey vowel harmony.

---

## 1. Definite vs. indefinite conjugation — decision procedure

1. **No direct object?** → **indefinite** (intransitive: *megyek, alszom, dolgozom*). Stop.
2. **Subject = "I" and object = "you" (*téged/titeket*)?** → special **-lak/-lek** form. Stop.
3. **Classify the object:**
   - **Definite** → definite (objective) conjugation: *a/az* + noun · proper name · possessive-suffixed noun · demonstrative *ezt/azt (a)…* · 3rd-person pronoun *őt, azt, őket, azokat* · reflexive *magát* / reciprocal *egymást* · *melyiket, mindegyiket, valamennyit, mind, az összeset* · a *hogy*-clause · formal *önt/önöket*.
   - **Indefinite** → indefinite (subjective) conjugation: no article · *egy* + noun · bare noun/plural · *sok/néhány/két/minden* + noun · *valamit/valakit* · *mit/kit* · *semmit/senkit* · 1st/2nd-person pronoun *engem/téged/minket/titeket*.
4. **Apply vowel harmony** to the chosen ending (back / front-unrounded / front-rounded).

> The subtle cell: **minden / mindenki / mindent / mindenkit → indefinite**; **mindegyik / valamennyi / mind / az összes → definite**; a **possessive suffix forces definite** even under a quantifier (*minden titkodat → ismerem*). (Coppock & Wechsler 2010.)

## 2. Present tense

Endings (back / front-unrounded / front-rounded):

| | indefinite | definite |
|---|---|---|
| én | -ok / -ek / -ök | -om / -em / -öm |
| te | -sz | -od / -ed / -öd |
| ő | –∅ | -ja / -i |
| mi | -unk / -ünk | -juk / -jük |
| ti | -tok / -tek / -tök | -játok / -itek |
| ők | -nak / -nek | -ják / -ik |

**lát** (see, back): *látok, látsz, lát, látunk, láttok, látnak* · **látom, látod, látja, látjuk, látjátok, látják**
**kér** (ask, front): *kérek, kérsz, kér, kérünk, kértek, kérnek* · **kérem, kéred, kéri, kérjük, kéritek, kérik**

Assimilation of the definite *-j-* to a stem-final sibilant (s, sz, z): *olvas* → **olvassa, olvassuk, olvassátok, olvassák**; *néz* → **nézi, nézzük, nézik**.

## 3. Past tense

Marker **-t-** for stems in *-l, -r, -n, -ny, -j, -ly, -s* and most others; **-ott/-ett/-ött** (long **-tt-**) for stems in two consonants, in *-ít*, and monosyllabic *-t* stems. Endings:

| | indefinite | definite |
|---|---|---|
| én | -tam / -tem | -tam / -tem |
| te | -tál / -tél | -tad / -ted |
| ő | -t / -ott / -ett / -ött | -ta / -te |
| mi | -tunk / -tünk | -tuk / -tük |
| ti | -tatok / -tetek | -tátok / -tétek |
| ők | -tak / -tek | -ták / -ték |

**vár** (wait): *vártam, vártál, várt, vártunk, vártatok, vártak* · **vártam, vártad, várta, vártuk, vártátok, várták**
**olvas** (read, *-ott* in 3sg indef.): *olvastam, olvastál, olvasott, olvastunk, olvastatok, olvastak* · **olvastam, olvastad, olvasta, olvastuk, olvastátok, olvasták**

> The 1sg past is identical in both conjugations (*vártam, kértem*); only context/object disambiguates.

## 4. The special I → you form (-lak / -lek)

Used only when the subject is 1sg and the object is 2nd person. **-lak** (back stem) / **-lek** (front stem).
- present: stem + -lak/-lek — **szeretlek, látlak, várlak, kérlek**
- past: 3sg-definite-past + -lak/-lek — **szerettelek, láttalak, vártalak**
- future: **foglak** + infinitive (*foglak látni*); conditional: **szeretnélek**; imperative: **szeresselek**

> Every *other* subject→you combination uses the **indefinite** conjugation (*Lát téged* "he sees you") — only 1sg→you gets -lak/-lek.

## 5. Conditional mood

Present conditional marker **-na/-ne** (indefinite) / **-ná/-né** (definite); **1sg indefinite is always -nék** (back *and* front).

**ír** (write, back): *írnék, írnál, írna, írnánk, írnátok, írnának* · **írnám, írnád, írná, írnánk, írnátok, írnák**
**szeret** (love, front): *szeretnék, szeretnél, szeretne, szeretnénk, szeretnétek, szeretnének* · **szeretném, szeretnéd, szeretné, szeretnénk, szeretnétek, szeretnék**
Irregular stems: *van → lennék/lenne*, *megy → mennék*, *eszik → ennék*, *tesz → tennék*.

**Past conditional** = indicative past + invariant **volna** (never inflected): *írtam volna, megettem volna, elmentünk volna.*

**ha (if) clauses:** real → *ha* + present (*Ha esik, otthon maradok*); unreal present → present conditional in **both** (*Ha lenne időm, elmennék*); counterfactual past → past conditional (*volna*) in **both** (*Ha lett volna pénzem, vettem volna*).

## 6. Imperative / subjunctive (felszólító mód)

Marker **-j-** + endings; one mood for commands and for embedded subjunctive (after *hogy, hadd, kell hogy*). Negation = **ne** (not *nem*).

**ír**: *írjak, írj / írjál, írjon, írjunk, írjatok, írjanak* · **írjam, írd / írjad, írja, írjuk, írjátok, írják** (2sg has a short and a long form; short is the default).

Coverb separates: *Olvasd el!* · *Menj be!* · negative *Ne olvasd el!*

**Assimilation of -j:**
- s + j → **ss**: *olvas* → olvasson · sz + j → **ssz**: *játszik* → játsszon · z + j → **zz**: *néz* → nézzen
- short-vowel/monosyllabic *-t*: t + j → **ss**: *fut* → fusson, *üt* → üssön, *lát* → lásson, *fizet* → fizessen
- *-ít* and long-vowel/cluster *-t*: t + j → **ts**: *segít* → segítsen, *tanít* → tanítson, *dönt* → döntsön, *ért* → értsen; *-st* cluster drops the t and doubles s: *fest* → fessen
- d (def. 2sg short) → **dd**: *ad* → add, *mond* → mondd

**Irregular imperatives:** *van/lesz* → légy/legyél, legyen; *jön* → **gyere** (2sg), jöjjön; *megy* → menj, menjen; *eszik* → egyél, egyen, def. edd; *iszik* → igyál, igyon, def. idd; *tesz/vesz/visz* → tegyél/vegyél/vigyél, def. tedd/vedd/vidd; *hisz* → **higgy**, def. hidd.

## 7. Future and the verb "to be"

- **fog** + infinitive: *fogok, fogsz, fog, fogunk, fogtok, fognak* (def. *fogom… fogják*). Coverb moves before *fog*: **El fogom olvasni.**
- **van** (be): present *vagyok, vagy, van, vagyunk, vagytok, vannak*; past *voltam, voltál, volt, voltunk, voltatok, voltak*. Dropped in 3rd-person present nominal predication (*Ő orvos*), kept for location/existence (*Itt van*).
- **lesz** (will be / become — future of *van*, never dropped): *leszek, leszel, lesz, leszünk, lesztek, lesznek*; past *lett-*.
- Negative existential: **nincs / nincsenek** (not *nem van*); under *sem* → *sincs/sincsenek*.

## 8. Modality and the inflected infinitive

**kell** (must) + dative + **inflected infinitive** (infinitive carries a personal ending; *kell* stays invariant):

| | -ni stem + suffix | *menni* | *olvasni* |
|---|---|---|---|
| 1sg | -nom / -nem / -nöm | mennem | olvasnom |
| 2sg | -nod / -ned / -nöd | menned | olvasnod |
| 3sg | -nia / -nie | mennie | olvasnia |
| 1pl | -nunk / -nünk | mennünk | olvasnunk |
| 2pl | -notok / -netek / -nötök | mennetek | olvasnotok |
| 3pl | -niuk / -niük | menniük | olvasniuk |

*El kell mennem.* (I have to go) · *Áginak el kell mennie.* · past *El kellett mennem.* · should *El kellene mennem.*

**Potential -hat/-het** (can / may / be allowed), attaches to the stem before endings: *vár → várhat*, *kér → kérhet*, *megy → mehet*, *lát → láthat*. Permission/possibility, vs. **tud** (acquired ability) and **képes** (capable).
Other modals: **lehet** (it is possible), **szabad** (permitted), **tilos** (forbidden), **muszáj** (must, colloquial), **szeretnék** (would like — conditional of *szeret*).

## 9. High-frequency irregular verbs (present indefinite unless noted)

| verb | present | notes |
|---|---|---|
| van (be) | vagyok, vagy, van, vagyunk, vagytok, vannak | past *volt-*; future *lesz* |
| megy (go) | megyek, **mész**, megy, megyünk, mentek, mennek | past *ment-*; imper. *menj* |
| jön (come) | jövök, jössz, jön, jövünk, jöttök, jönnek | past *jött-*; imper. *gyere* |
| eszik (eat) | eszem, eszel, eszik, eszünk, esztek, esznek | past 3sg **evett**; inf. *enni* |
| iszik (drink) | iszom, iszol, iszik, iszunk, isztok, isznak | back endings; past 3sg **ivott**; inf. *inni* |
| tesz/vesz/visz/hisz | teszek/veszek/viszek/hiszek… | past **tett-/vett-/vitt-/hitt-** |

**ikes verbs** (3sg in *-ik*: *lakik, dolgozik, alszik, eszik, iszik*): standard written 1sg is **-m** (*lakom, dolgozom, eszem*); the colloquial *-k* (*lakok, eszek*) is stigmatized in writing — prefer **-m** in formal output.

## 10. Case suffixes (the inventory)

The three-way spatial system (on *ház* "house", back / *kert* "garden", front):

| | static (*hol?*) | to (*hová?*) | from (*honnan?*) |
|---|---|---|---|
| **interior** | -ban/-ben — házban | -ba/-be — házba | -ból/-ből — házból |
| **surface** | -on/-en/-ön — házon | -ra/-re — házra | -ról/-ről — házról |
| **proximity** | -nál/-nél — háznál | -hoz/-hez/-höz — házhoz | -tól/-től — háztól |

Other cases:

| case | suffix | use | example |
|---|---|---|---|
| accusative | -t (link -ot/-at/-et/-öt) | direct object | *házat, kertet, almát* |
| dative | -nak/-nek | to/for; possessor | *Péternek* |
| instrumental | -val/-vel | with / by means of | *késsel, vonattal* |
| causal-final | -ért | for the sake of / to get | *pénzért* |
| translative | -vá/-vé | into / becoming | *kővé, sóvá* |
| terminative | -ig | until / for (duration) | *hat óráig, három évig* |
| essive-formal | -ként | as (in the role of) | *tanárként* |
| temporal | -kor (invariant) | at (clock time/occasion) | *ötkor, karácsonykor* |

**Place names:** Hungarian towns + "Hungary" + islands take the **surface** series (*Budapesten/-re/-ről, Magyarországon, Tajvanon* "on Taiwan island"); foreign cities/countries take **interior** (*Londonban, Németországban, Tajvanban* "in Taiwan country"). Minimal pair: *Tolnán* (town) vs. *Tolnában* (county).

**Time:** clock → **-kor** (*ötkor*); days → superessive **-n** (*hétfőn, pénteken*; but *vasárnap* bare); months/years → inessive (*januárban, 2020-ban*); parts of day → bare adverb (*reggel, délután, este*); "in X time" → *múlva* (*két hét múlva*); "every day" → *naponta*.

**-val/-vel and -vá/-vé assimilation:** after a vowel the *v* stays (*autóval, kővé*); after a consonant the *v* assimilates and the consonant doubles — *busz → busszal, kés → késsel, vonat → vonattal, víz → vízzel, ember → emberré, bolond → bolonddá*.

## 11. Postpositions (separate words, after the noun)

**Directional triples** (static / to / from), all take the bare nominative:

| meaning | static | to | from |
|---|---|---|---|
| under | alatt | alá | alól |
| above | fölött/felett | fölé | fölül |
| behind | mögött | mögé | mögül |
| in front of | előtt | elé | elől |
| beside | mellett | mellé | mellől |
| between/among | között | közé | közül |

**Other postpositions with the bare nominative:** *után* (after), *körül* (around), *miatt* (because of), *helyett* (instead of), *szerint* (according to), *ellen* (against), *nélkül* (without), *által* (by — agent), *felé* (toward), *óta* (since), *múlva* (in [time]), *közben / során / alatt* (during).

**Postpositions that govern a case:** *együtt*, *szemben* (+ -val/-vel); *át, keresztül, túl, belül, kívül* (+ superessive -n); *ellenére, megfelelően, számára* (+ -nak/-nek); *képest* (+ -hoz/-hez/-höz).

**Personal forms** (postposition + possessive ending, not *postposition + pronoun*): *alattam* (under me), *mögötte* (behind it), *mellettünk* (beside us), *előttem* (in front of me), *köztük* (among them), *utánad* (after you), *nélküle* (without him/her), *szerintem* (in my opinion), *miattad* (because of you), *helyettem* (instead of me), *ellened* (against you).

## 12. Demonstrative declension (agrees with the noun; *-z* assimilates)

The demonstrative takes the **same case** as the head noun, the article *a/az* stays between them, and the *-z* of *ez/az* assimilates before a consonant.

| case | this | that | example |
|---|---|---|---|
| accusative | ezt | azt | *ezt a könyvet* |
| inessive (in) | ebben | abban | *ebben a házban* |
| illative (into) | ebbe | abba | *abba a városba* |
| elative (out of) | ebből | abból | |
| superessive (on) | ezen | azon | *ezen az asztalon* |
| sublative (onto) | erre | arra | |
| delative (off/about) | erről | arról | *arról a filmről* |
| adessive (at) | ennél | annál | |
| allative (to) | ehhez | ahhoz | |
| ablative (from) | ettől | attól | |
| dative | ennek | annak | *annak a háznak…* |

Plurals (*ezek/azok*) do **not** assimilate: *ezekben a házakban, azokat a könyveket.*

## 13. Possession

**Singular possessed** (one thing owned) — *ház* / *könyv*:

| | -m form | *ház* | *könyv* |
|---|---|---|---|
| 1sg | -m / -om/-em/-öm | házam | könyvem |
| 2sg | -d / -od/-ed/-öd | házad | könyved |
| 3sg | -a/-e / -ja/-je | háza | könyve |
| 1pl | -unk/-ünk | házunk | könyvünk |
| 2pl | -otok/-etek/-ötök | házatok | könyvetek |
| 3pl | -uk/-ük / -juk/-jük | házuk | könyvük |

The 3sg *-a/-e* vs *-ja/-je* split is lexical: *ház → háza, ablak → ablaka* but *kert → kertje, barát → barátja.*

**Plural possessed** (several things owned) — marked by **-i-**, not the plural *-k*: *könyv* → **könyveim, könyveid, könyvei, könyveink, könyveitek, könyveik**; *ház* → **házaim … házaik**. (Never *könyvekem*.)

**Order:** stem + possessive + (plural) + case — *házamban* (in my house), *házaimban* (in my houses).

**Possessor-first NP:** *Péter könyve* (Peter's book), *a ház ablaka* (the window of the house). **Dative possessor** (emphasis/disambiguation, obligatory with demonstratives): *Péternek a könyve*, *annak a háznak az ablaka*.

**"Have"** = dative possessor + *van/nincs* + possessed-with-suffix: *Van egy kutyám* (I have a dog), *Nincs pénzem* (I have no money), *Péternek van autója* (Peter has a car), *Vannak barátaim* (I have friends), *A gyerekeknek nincsenek autóik* (the children have no cars).

**The -é form** ("the one belonging to X", possessed noun dropped): *Ez a könyv Péteré* (This book is Peter's); *Kié ez? — Péteré.*; plural *-éi*: *Istvánéi* (István's ones).

## 14. Pronouns

- **Personal:** én, te, ő, mi, ti, ők (*ő* is genderless). Pro-drop: omit as subject unless emphatic.
- **Formal address:** *ön/önök* (neutral-polite), *maga/maguk* (familiar, can be cold) — both with **3rd-person** verbs.
- **Reflexive:** magam, magad, maga, magunk, magatok, maguk (*Megvágtam magam* — I cut myself). Emphatic "own" = *saját* (*a saját házam*).
- **Possessive pronouns** ("mine, yours"): enyém, tiéd, övé, miénk, tiétek, övék — with the article: *az enyém, a tiéd*.
- **Relative:** **aki** (people), **ami** (things / whole-clause antecedent; colloquial), **amely** (things; literary), **amelyik** (which one, selecting from a set). A comma always precedes the relative clause.

## 15. Number, adjectives, comparison

- **Plural -k** with linking vowel by harmony: *ház → házak, ember → emberek, könyv → könyvek, ablak → ablakok, gyümölcs → gyümölcsök*; vowel-final adds bare -k and lengthens final *-a/-e*: *macska → macskák, fürdő → fürdők*.
- **After a numeral/quantifier → singular noun:** *két macska, öt fiú, sok ember, minden gyerek, néhány diák.*
- **Adjective agreement:** attributive does **not** agree (*piros almák, nagy házak*); predicative **does** (*Az almák pirosak; A házak nagyok*). Predicative plural endings: *piros → pirosak, nagy → nagyok, szép → szépek, kicsi → kicsik.*
- **Comparative -bb/-abb/-ebb; superlative leg- + comparative** (with *a/az*): *gyors → gyorsabb → a leggyorsabb*; extreme *legesleg-*. Irregulars: *jó→jobb, nagy→nagyobb, sok→több, kevés→kevesebb, kicsi/kis→kisebb, szép→szebb, hosszú→hosszabb, könnyű→könnyebb, nehéz→nehezebb, rossz→rosszabb.*
- **"Than"** = adessive **-nál/-nél** (*magasabb a nőnél; magasabb nálam*) **or** **mint + nominative** (*magasabb, mint én* — never *mint engem*). Don't combine the two.
- **"As…as"** = **olyan … mint** (*olyan magas, mint én*); "the same as" = *ugyanolyan … mint*; amounts = *annyi / ugyanannyi … mint*.

## 16. Vowel harmony

- **Classes:** back = a á o ó u ú; front-unrounded = e é i í; front-rounded = ö ő ü ű.
- **Rule:** any back vowel in the stem → back suffixes; only front vowels → front suffixes; the **last vowel** decides mixed stems. *i, í, e, é* are **neutral/transparent**.
- **Two-form suffixes** (back/front only): -ban/-ben, -ba/-be, -ból/-ből, -nál/-nél, -tól/-től, -ról/-ről, -nak/-nek, -val/-vel, -ig, -ért.
- **Three-form suffixes** (add front-rounded): -hoz/-hez/-höz, -on/-en/-ön, -ok/-ek/-ök (plural link), -ra/-re uses two forms. *gyümölcshöz, földön, ökrök.*
- **Antiharmonic stems** look front but take **back** suffixes — memorize: *híd → hidak, híd → hídban; férfi → férfiak; cél → célok, célból; szid; nyíl → nyilak.* Contrast normal front stems *szék → székek, szív → szívek.*
