# Hungarian style, formatting, and lexicon (reference)

Full conventions and word lists backing `../SKILL.md`. The high-frequency rules are already in SKILL.md; this file is for completeness and lookup.

---

## 1. Register, greetings, and politeness

| Form | Pronoun | Verb | When |
|---|---|---|---|
| informal | te / ti | 2nd person | friends, peers, children, family, animals; youth/tech/casual |
| formal (safe default) | ön / önök | **3rd person** | strangers, customers, officials, elders, business |
| familiar-formal | maga / maguk | 3rd person | **avoid in formal writing** — can be cold, distant, or rude; never to an elder/superior |
| deferential | (none) — *tetszik* + infinitive | — | warmth to elders; *Hogy tetszik lenni?*, *Mit tetszik kérni?* |

- A formal pronoun takes a **3rd-person** verb: *Ön beszél magyarul* (not *Ön beszélsz*); *Önök tudnak/tudják*.
- Keep one register through the whole text — pronouns, verbs, possessives, and imperatives must all match. English "you" repeated in a document maps to **one** Hungarian register.

**Greetings / sign-offs:**

| Register | Opening | Closing |
|---|---|---|
| informal | Szia / Sziasztok; Szervusz(tok) | Üdv; Puszi; Szia |
| spoken neutral-formal | Jó napot (kívánok)!; Jó reggelt/estét kívánok! | — |
| formal letter/email | **Tisztelt Cím!**; Tisztelt Uram! / Hölgyem!; **Tisztelt Kovács Úr! / Kovács Tanár Úr!** | **Tisztelettel,** / **Üdvözlettel,** / Maradok tisztelettel |
| semi-formal | Kedves Anna! / Kedves Kovács Úr! | Üdvözlettel; Köszönettel; Üdv |

- **Salutations end with "!"**, not a comma: **Tisztelt Kovács Úr!** (then the body starts capitalized on the next line). *Kedves X* may take "!" or comma.
- *Tisztelt* + full name (*Tisztelt Kovács János!*) reads as over-formal/impolite — use title + surname.
- *drága* means "dear" in the *emotional/expensive* sense; don't use it for letter salutations.

**Imperative softening** (a bare command to *ön* is abrupt): use the conditional, *legyen szíves*, *kérem*, *tudna* + infinitive, *tessék* + infinitive.
- Close the window. → **Becsukná az ablakot?** / **Legyen szíves becsukni az ablakot.**
- Please send the file. → **Kérem, küldje el a fájlt.** / **Meg tudná küldeni a fájlt?**
- Sit down (to an elder). → **Tessék helyet foglalni.** / **Foglaljon helyet.**

## 2. Punctuation

- **Quotation marks:** **„…”** (low-opening U+201E, high-closing U+201D) — *She said: "I'm coming."* → **Azt mondta: „Jövök.”** (closing punctuation inside the quote for a full quoted sentence). Nested = reversed guillemets **»…«**: **„egy »idézet« belül”**. Never English "…" or '…'.
- **Comma before subordinate clauses — always**, even where English omits it: before **hogy** (*Azt gondolom, hogy jó.*) and before **aki / ami / amely / mert / ha / mint** (*az ember, aki jött*; *Maradok, ha hívsz.*).
- **Comma with *és / s / meg / vagy*:** **none** when joining items in a list or simple coordinated phrases (*kenyér, tej és tojás*); a comma **is** usual when *és/vagy* joins two full independent clauses (*Bevágta az ajtót, és elrohant.*).
- **Colon, dash, ellipsis** behave broadly as in English; the dash is the long *gondolatjel* (–) with spaces.

## 3. Capitalization (much lighter than English)

Lowercase in Hungarian (uppercase in English):

| category | English | Hungarian |
|---|---|---|
| months | January | január |
| days | Monday | hétfő |
| languages | English | angol |
| nationality/adjective | a German engineer | egy német mérnök |
| holidays | Christmas | karácsony |
| "I" | I think | (én) gondolom |

- **Headings/titles:** sentence case (capitalize only the first word + proper nouns): *Privacy Policy* → **Adatvédelmi irányelvek**. Established work/institution titles capitalize key words (*A Gyűrűk Ura*).
- **Letters:** the polite **Ön / Önt / Önnek / Önök** is capitalized as a courtesy; *Te/Ti* may be capitalized in personal letters (increasingly lowercase).

## 4. Dates, times, numbers

- **Dates descend, with periods:** *June 13, 2026* → **2026. június 13.** (lowercase month, trailing period). Numeric: **2026. 06. 13.** Abbrev.: **2026. jún. 13.**
- **"On [date]"** — add the locative suffix and **drop the period** before the hyphen: **2026. június 13-án.** Day suffixes: 1 → **1-jén** (elsején), 2 → 2-án, 3 → 3-án, 4 → 4-én, 15 → 15-én, 20 → 20-án, 21 → 21-én, 31 → 31-én.
- **Decimal = comma; thousands = space (or period), grouped by 3:** `1,234.56` → **1 234,56**; `3.5%` → **3,5%**.
- **Currency:** unit follows after a space: **1 000 Ft**, **1 500,50 Ft**, **20 €** (or *20 EUR*). With a suffix, hyphenate: **5%-kal** (by 5%).
- **Large numbers (long scale):** million = **millió** (10⁶); **billion = milliárd** (10⁹); **trillion = billió** (10¹²); quadrillion = billiárd (10¹⁵). *$5 billion* → **5 milliárd dollár** (using *billió* here is 1000× wrong — critical in finance).
- **Time:** 24-hour clock, separator period or colon: **14.30 / 14:30**; colloquial **fél 8** = 7:30, **negyed 8** = 7:15, **háromnegyed 8** = 7:45. "At 5" → **5-kor / ötkor**.
- **Units:** space between number and unit: **20 °C, 5 kg, 100 km/h, 50 m²**. **Ordinals:** number + period: **3.** (harmadik), **XXI. század** (21st century).

## 5. Suffixes on numerals, acronyms, foreign words

Attach via a **hyphen**, harmonized, with doubling by pronunciation: **5-kor** (at 5), **10-én** (on the 10th), **20-as** (the "20" one), **3%-kal**, **EU-ban / ENSZ-ben** (in the EU / UN), **Windows-szal** (with Windows). Common abbreviations take a period: **stb.** (etc.), **pl.** (e.g.), **kb.** (approx.), **dr.**, **u.** (utca).

## 6. False friends (extended)

English → tempting WRONG cognate (its real meaning) → CORRECT Hungarian.

| English | wrong cognate (means) | correct |
|---|---|---|
| actual (real) | aktuális (current, topical) | tényleges, valódi |
| actually | aktuálisan | valójában, tulajdonképpen |
| current | — | aktuális, jelenlegi, mostani |
| eventually | eventuális (possible, contingent) | végül, végül is |
| sympathetic | szimpatikus (likeable) | együttérző, megértő |
| sympathize | szimpatizál (be fond of, side with) | együttérez |
| sensible | szenzibilis (sensitive) | értelmes, józan, ésszerű |
| sensitive | (szenzitív/érzékeny is fine) | érzékeny |
| realise (understand) | realizál (implement, book a profit) | rájön, felismer |
| novel (book) | novella (short story) | regény |
| short story | — | novella, elbeszélés |
| concrete (material) | konkrét (specific, definite) | beton |
| concrete (specific) | — | konkrét |
| map | mappa (folder) | térkép |
| massive | masszív (sturdy, solid) | óriási, hatalmas |
| solid | szolid (modest, restrained) | szilárd, masszív, megbízható |
| simple | szimpla (single, plain) | egyszerű |
| receipt | recept (recipe; prescription) | nyugta, blokk |
| sodium | szóda (soda/washing soda) | nátrium |
| transparent | transzparens (banner, placard) | átlátszó |
| genie | zseni (genius) | dzsinn |
| genius | — | zseni, lángész |
| smoking (verb) | szmoking (dinner jacket) | dohányzás |
| fabric | fabrika (archaic: factory) | szövet, anyag, textília |
| factory | — | gyár, üzem |
| gymnasium / grammar school | gimnázium = academic secondary school | gym = tornaterem, edzőterem |
| apartment | apartman (holiday/serviced suite) | lakás |
| pension (money) | panzió (guesthouse) | nyugdíj |
| conserve (preserve) | konzerv (tinned food) | megőriz, megóv |
| confection (sweet) | konfekció (ready-made clothing) | édesség, desszert |
| billion | billió (= trillion, 10¹²) | milliárd |
| trillion | — | billió |
| to advertise | reklamál (to complain) | reklámoz, hirdet |
| action / sale | akció (often = promotion/sale) | cselekvés (action); akció/leárazás (sale) |

> *kontroll/kontrollál* leans "check / monitor / follow-up" (medical *kontroll* = check-up); plain "control" is usually **irányít, szabályoz, ellenőriz, kézben tart** — choose by context.

## 7. Light-verb collocations and calqued clichés

English "make/take/do + noun" rarely maps to *csinál*; pick the right light verb:

| English | calque (wrong) | native |
|---|---|---|
| make a decision | döntést csinál | döntést hoz / dönt |
| take a photo | fotót vesz | fényképet készít / fotót csinál / fényképez |
| make a mistake | csinál egy hibát | hibát követ el / hibázik |
| make a phone call | csinál egy hívást | telefonál / felhív |
| make a promise | csinál ígéretet | ígéretet tesz |
| ask a question | kérdez egy kérdést | kérdést tesz fel / kérdez |
| set up a company | csinál egy céget | céget alapít / hoz létre |
| spend time | időt költ | időt tölt |
| spend money | — | pénzt költ |
| do a favor | csinál egy szívességet | szívességet tesz |
| make sense | értelmet csinál | van értelme / logikus |
| have breakfast | van reggeli | reggelizik |
| take a shower | vesz egy zuhanyt | (le)zuhanyozik |

**Social / email formulas** (fixed, not literal):

| English | calque | native |
|---|---|---|
| Have a nice day! | Legyen szép napod! | Szép napot (kívánok)! |
| How are you? (formal) | Hogy vagy? | Hogy van? (elder: Hogy tetszik lenni?) |
| Nice to meet you. | Jó találkozni veled | Örvendek. / Örülök, hogy megismerhetem. |
| I look forward to … | Előre nézek a…-hoz | Várom a… / Előre is köszönöm. |
| Please find attached… | Kérlek találd csatolva | Mellékelten küldöm… / Csatoltan megküldöm… |
| Thank you in advance | Köszönöm előre | Előre is köszönöm. |
| Looking forward to hearing from you | Várva hallani tőled | Várom mielőbbi válaszát. |
| I'm good (no thanks) | Jó vagyok | Köszönöm, megvagyok. / Köszönöm, nem kérek. |

**Loan/anglicism overuse:** prefer Hungarian — *feedback* → **visszajelzés**, *update* → **frissítés**, *support* → **támogat / ügyfélszolgálat**, *challenge* (verb) → **megkérdőjelez / próbára tesz** (noun *kihívás* is fine).

**Drop redundant pronouns/possessives:** *I washed my hands* → **Megmostam a kezem** (not *Én megmostam az én kezeimet*); *She loves her job* → **Szereti a munkáját**; *Do you have your ticket?* → **Megvan a jegyed?**
